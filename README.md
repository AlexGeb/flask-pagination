# get started

Clone this repo :

`git clone https://github.com/AlexGeb/flask-pagination.git`

Install dependencies :

`cd flask-pagination`

`pip install -r requirements.txt`

Start the Flask app in dev mode :

`python backend/app.py`

Then for the front :

`cd front`

`yarn install`

`yarn start`
